One Line Installation

Understandably, setting up a server from scratch is a pain. install.chae.sh assists you with this. You can select the packages you wish to install along with a variety of other tasks and it will generate a one-line command to complete the task for you. It can also handle various other tasks such as adding the correct PPAs, updating the system and downloading backups. This project is open source and can be found on GitHub. Please submit a pull request if you add anything, we'd like to implement it!
